# AmountType

The amount type of the deduction for the pay period. Only present for unprocessed payrolls.

## Example Usage

```java
import com.gusto.embedded_api.models.components.AmountType;

AmountType value = AmountType.FIXED;
```


## Values

| Name      | Value     |
| --------- | --------- |
| `FIXED`   | fixed     |
| `PERCENT` | percent   |
