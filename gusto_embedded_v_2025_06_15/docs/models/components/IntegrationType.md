# IntegrationType

The `integration_type` used for the report, if `aggregation` was 'integration.' Otherwise, this will be null.

## Example Usage

```java
import com.gusto.embedded_api.models.components.IntegrationType;

IntegrationType value = IntegrationType.XERO;
```


## Values

| Name   | Value  |
| ------ | ------ |
| `XERO` | xero   |
| `QBO`  | qbo    |
