# PolicyType

Type of the time off policy. Currently only "vacation" and "sick" are supported

## Example Usage

```java
import org.openapis.openapi.models.operations.PolicyType;

PolicyType value = PolicyType.VACATION;
```


## Values

| Name       | Value      |
| ---------- | ---------- |
| `VACATION` | vacation   |
| `SICK`     | sick       |
