# GrantType

Set system_access to create a system access token, refresh_token to refresh an existing token

## Example Usage

```java
import org.openapis.openapi.models.operations.GrantType;

GrantType value = GrantType.REFRESH_TOKEN;
```

| Name            | Value           |
| --------------- | --------------- |
| `REFRESH_TOKEN` | refresh_token   |
